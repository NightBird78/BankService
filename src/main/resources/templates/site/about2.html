А теперь немного о проекте

  • Зачем?

    Проект сделан что-бы обуздать каждый уголок джавы, спринга и баз даных естественно ничто некогда не идет гладко так сказать
    не имеет багов тот - кто нечего не делает
    Проект имеет API банка, для того чтобы узнать текущий курс валют,
    Swagger (OpenAPi) что бы в любой момент можно было зайти и посмотреть что просит каждый контроллер,
    BootStrap что-бы каждый мог посмотреть,
    не на классический html а на оформленную страницу
